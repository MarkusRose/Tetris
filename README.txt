This is a very basic old school, text based tetris game for the console.
The starting source was a youtube video "Code-It-Yourself! Tetris - Programming from Scratch (Quick and Simple C++)" by javidx9.

Controls:
Left Key -> Move tetromino to the left
Right Key -> Move tetromino to the right
Down Key -> Move tetromino down
Up Key -> Rotate tetromino
